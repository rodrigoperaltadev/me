# Getting Started with the Admin Panel

The admin panel provides a terminal-inspired interface for managing portfolio content.

## Access

Navigate to `/admin` and authenticate with your password.

## Features

- **Experience Management**: Create, edit, and delete professional experience entries with multi-lingual support (EN/ES).
- **Blog Engine**: Publish technical articles using MDX format with draft/published status.
- **Security**: All admin routes are protected with signed cookies and noindex directives.

## Writing Posts

Use the Blog section to create new posts. The editor supports:
- Markdown syntax
- Frontmatter metadata (title, date, tags, description)
- Draft mode for unpublished content

Posts are stored as `.mdx` files in `content/blog/`.
